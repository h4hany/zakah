Usage Guidelines

Recommended Uses

  • Floating widget buttons (64px-128px)
  • Mobile app icons (1024px iOS, 512px Android)
  • Favicons (16px, 32px, 64px)
  • PDF report headers (128px-256px)
  • Website logos and navigation
  • Marketing materials and presentations

Technical Specs

  • Format: SVG (scalable vector)
  • Viewbox: 64x64 (1:1 aspect ratio)
  • Stroke weight: 2-3px (consistent)
  • Colors: #C9A24D, #212325, #fff
  • Transparent background
  • Min size: 16px, Max: unlimited
